Welcome to Tracing Screen

Getting Started

This tool helps you trace images with precision. Let's learn how to use it!

First, upload an image using the upload button at the bottom of the screen.

Adjusting Your Image

Once your image is loaded, you can adjust its appearance:

  • Use the sliders to change contrast and brightness
  • Adjust opacity to make tracing easier
  • Transform the image with rotation and zoom

Using the Grid

The grid helps with precise tracing:

  • Toggle the grid with the grid button
  • Customize grid color and opacity in Theme Settings

Saving Your Work

You can save your configurations as presets:

  • Click the presets button
  • Name your configuration and save it
  • Load it anytime for quick setup

Keyboard Shortcuts

Speed up your workflow with these shortcuts:

  • Ctrl+Z: Undo
  • Ctrl+Y: Redo
  • G: Toggle Grid
  • L: Lock/Unlock Interface
Reset All Settings?
This will reset all adjustments, remove the current image, and return to default settings. This action cannot be undone.
